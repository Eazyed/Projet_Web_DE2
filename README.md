# Projet_Web_DE2
Repo du projet web pour le cours de Technologies du Web, Promo DE2 2019/2020

## Pré-requis

Ce projet utilise le framework Vue.js qui est basé sur node.js. Il est donc nécessaire d'avoir installé une version récente de nodejs/npm
Téléchargement : https://nodejs.org/en/

### Lancer le projet

Après avoir cloné le repo en local, on positionne un terminal dans le dossier projet_web qui contient le fichier package.json, on lance alors la commande

```
npm install
```

Ce qui va nous permettre de récupérer les packages nodejs utilisés pour ce projet.

On lance le serveur de développement avec la commande suivante :

```
npm run serve
```

On accède à la page web par le port 8080 sur localhost
